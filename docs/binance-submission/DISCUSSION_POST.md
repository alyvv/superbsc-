# Introducing MEFAI: Open-Source Crypto Intelligence Terminal Powered by All 7 Skills Hub APIs

Hey Binance community!

We built **MEFAI** — an open-source, keyboard-driven crypto intelligence terminal that uses **every single Binance Skills Hub API** in a unified workflow. Think Bloomberg Terminal for crypto, but free and open source.

## What Makes It Different

**1. All 7 Skills in One Place**
No other project connects all 7 APIs. MEFAI does — from spot trading to meme discovery to security audits to smart money tracking.

**2. Auto-Scanner (Unique Feature)**
A background engine that continuously:
- Pulls new tokens from Meme Rush
- Audits each one for security
- Checks if smart money is buying
- Cross-references with trading signals
- Scores and surfaces the most promising opportunities

No other tool does this automated multi-API intelligence.

**3. Terminal-First**
Keyboard-driven, command palette, layout presets. Every action has a shortcut. No mouse required for power users.

**4. Zero Framework**
Pure vanilla JS + Web Components. No React, no Vue, no build step. Clone it, open it, it works.

**5. Plugin Architecture**
Add a panel in ONE file. Under 100 lines. No config changes.

## 18 Panels

Every panel maps to a specific Skill Hub API. The Auto-Scanner uses all 7 simultaneously.

## Tech Stack

- Frontend: Vanilla JS + Web Components
- Backend: Python FastAPI (thin proxy)
- Charts: TradingView lightweight-charts
- Deploy: Docker Compose
